# gd-killer

一个基于DrissionPage的京东秒杀python脚本

# 安装环境

1. 自行安装谷歌浏览器
2. 自行安装python
3. 使用 pip 安装 DrissionPage：`pip install DrissionPage`

# 使用

修改变量`killer_time`为你的秒杀时间，直接用python运行即可(需要提前选择好你要购买的商品型号并添加到京东购物车以及设置好默认的发货地址等信息)

```python
python gd_killer.py
```
